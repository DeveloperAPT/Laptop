Ключевые критерии выбора

Разберём 6 главных параметров, на которые стоит обратить внимание перед покупкой

На что смотреть в первую очередь

Каждый критерий влияет на комфорт использования

Процессор (CPU)

«Мозг» ноутбука. Определяет скорость выполнения задач. Для офисных задач хватит Intel Core i3 / AMD Ryzen 3. Для учёбы и multitasking — i5 / Ryzen 5. Для игр и монтажа — i7 / Ryzen 7 и выше.

Совет: не гонитесь за i9 — для 95% задач достаточно i5/Ryzen 5.

Оперативная память (ОЗУ)

Влияет на многозадачность. 4 ГБ — абсолютный минимум (тормоза гарантированы). 8 ГБ — комфортный стандарт для учёбы и работы. 16 ГБ — для игр, видеомонтажа, виртуальных машин.

Совет: выбирайте модель с возможностью расширения ОЗУ.

Накопитель (SSD / HDD)

SSD обеспечивает быструю загрузку системы и программ. Минимум — 256 ГБ SSD. Оптимально — 512 ГБ. HDD в 2025 году — только как дополнительный диск для хранения файлов.

Совет: никогда не берите ноутбук только с HDD — это прошлый век.

Экран

Диагональ: 13–14″ для мобильности, 15.6″ для универсальности, 17″ для игр. Разрешение: минимум Full HD (1920×1080). IPS-матрица — обязательна для комфортных углов обзора.

Совет: для дизайнеров ищите 100% sRGB и разрешение выше Full HD.

Автономность

Ёмкость батареи измеряется в Вт·ч (Wh). 40–50 Wh — 4–6 часов. 60–75 Wh — 8–10 часов. 80+ Wh — 10+ часов. Учитывайте, что игровые ноутбуки живут 3–5 часов даже с большой батареей.

Совет: для учёбы берите ноутбук с батареей от 60 Wh и USB-C зарядкой.

Клавиатура и порты

Хорошая клавиатура с подсветкой — must-have для набора текстов. Порты: минимум 2×USB-A, 1×USB-C, HDMI, разъём для наушников. Проверьте наличие слота для SD-карты, если вы фотограф.

Совет: перед покупкой попробуйте печатать на клавиатуре в магазине.
